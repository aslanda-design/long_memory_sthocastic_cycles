# Changelog

All notable changes to this project are documented here.

The format follows Keep a Changelog conventions, and this project uses
semantic versioning while the public API stabilizes.

## [0.1.0] - 2026-07-08

### Added

- First publishable release of `cyclical-fractional-test`.
- Public API for periodograms, autocorrelograms, candidate evaluation, diagnostics,
  and the full cyclical fractional long-memory test.
- Adaptive and fixed-grid search over fractional integration parameter `D`.
- Single-cycle and aggregate multi-cycle candidate evaluation.
- White-noise, AR(1), and AR(2) residual error specifications.
- Scikit-learn-style `CyclicalFractionalModel` with `fit`, `predict`,
  recursive prediction, and prediction intervals.
- Mathematical notes, data-flow diagrams, implementation notes, and publishing
  documentation.
